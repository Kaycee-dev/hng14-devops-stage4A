## Why mermaid, not PNG

- dev.to renders mermaid blocks natively.
- GitHub renders mermaid in markdown previews.
- Source files == final renderings; no PNG export step that could rot.
- Diagrams stay editable forever instead of becoming binary blobs that no
  one updates.
